# Projeto de Ciência de Dados - 2025/2026

## Descrição
Projeto desenvolvido no âmbito da unidade curricular de Ciência de Dados (3º ano de Engenharia Informática). O objetivo principal é aplicar o ciclo de vida analítico a um problema de [inserir o tipo de problema: classificação/regressão/etc.].

## Estrutura do Repositório
* `01_planos_objetivos/`: Enunciado e planeamento.
* `03_codigo/`: Jupyter Notebooks com a Análise Exploratória (EDA) e modelação.
* `04_outputs/`: Gráficos e resultados gerados.
* `05_relatorio/`: Relatório do projeto.

*Nota: A pasta de datasets não está incluída no repositório por motivos de tamanho/privacidade.*
